# Full Figma Design Request — Rasid Executive Platform Redesign

I want to redesign an Arabic RTL platform called **Rasid / راصد**.

Rasid is a national-grade privacy policy monitoring and compliance intelligence platform. It scans websites, analyzes privacy policies, measures compliance, displays dashboards and analytics, manages advanced scans, and includes an AI assistant called **Rasid**.

The current UI is functional, but the visual design is not at the required level. I do not want a simple color refresh. I want a major professional redesign that moves the platform to a completely higher visual level.

The final design must feel like:

**Executive / Premium / Government-grade / National compliance intelligence platform**

It must not look like a generic SaaS dashboard.

# 3. Language and Direction

The entire product is Arabic and RTL.

The design must be RTL by default.

Required RTL behavior:

1. Sidebar on the right.
2. Page content flows from right to left.
3. Main titles align right.
4. Paragraphs align right.
5. Search fields align right.
6. Filters start from the right.
7. Tabs start from the right.
8. Tables start from the right.
9. First logical column is on the right.
10. Actions are positioned logically for Arabic UI.
11. Modals are RTL.
12. Dropdowns are RTL.
13. Tooltips are RTL.
14. Chart legends should support RTL as much as possible.
15. No page should feel like an English LTR dashboard translated into Arabic.

# 4. Typography

Use Arabic typography suitable for a serious executive platform.

Preferred font:

**Tajawal**

Use it across the interface.

If another Arabic font is needed, it must be modern, clean, and highly readable.

Typography requirements:

1. Clear hierarchy.
2. Large readable page titles.
3. Strong KPI numbers.
4. Clean table text.
5. Comfortable line-height.
6. Avoid overly thin font weights.
7. Use bold weights for key metrics.
8. Use medium weights for section titles.
9. Use regular weights for descriptions.
10. Use Arabic numerals or consistent number styling depending on the visual system.

Suggested typography scale:

* Page title: 28–36 px.
* Section title: 20–24 px.
* Card title: 15–18 px.
* KPI numbers: 34–56 px.
* Body text: 14–16 px.
* Table text: 13–15 px.
* Caption/helper text: 12–13 px.

The typography must feel premium and clear, not crowded.
