Enhance the current platform UI design with an ultra-premium cinematic motion direction while preserving the existing structure, content hierarchy, layout purpose, and brand identity.

The goal is not to redesign the platform from scratch, but to elevate the current design into a more advanced, intelligent, high-end, AI-driven visual experience.

## Core Requirements

Apply the following improvements across all platform pages:

### 1. Animated Logo System

Create a refined animated logo treatment for all pages.

The logo animation must feel:

* Premium
* Smooth
* Intelligent
* Minimal
* Technological
* Professional
* Not distracting

Suggested motion direction:

* Soft glow pulse
* Subtle gradient movement
* Elegant light sweep
* Gentle particle shimmer
* Micro-scale breathing motion
* Smooth reveal on page load

The logo animation must not feel childish, excessive, or noisy. It should communicate AI, data intelligence, national digital transformation, trust, and advanced technology.

The backgrounds should express:

* Artificial intelligence

* Privacy monitoring
* Digital governance
* Cyber trust
* Smart analytics

* Ultra-premium SaaS quality

Use subtle cinematic background elements such as:

* Animated gradient fields

* Data-flow particles
* Glassmorphism depth layers
* Slow-moving mesh gradients
* Minimal grid systems
* Digital light trails
* Floating geometric shapes
* Privacy/security-inspired visual patterns
* Elegant depth blur effects

The backgrounds must be:

* Different from page to page
* Smoothly animated
* Professional
* Non-distracting
* Low contrast behind text areas
* Optimized for readability
* Visually premium
* Consistent with the platform identity

Avoid heavy, aggressive, gaming-style, neon-overloaded, or visually noisy backgrounds.

### 6. Motion Quality Standards

All animations must follow these rules:

* Smooth and premium
* Not excessive
* Not distracting
* Not too fast
* Not too slow
* No harsh movement
* No random chaotic animation
* No visual clutter
* No readability problems
* No layout shifting
* No unnecessary complexity

Use refined timing:

* Page entrance: 500–900ms
* Card entrance: 300–600ms
* Hover transition: 180–300ms
* Background motion: very slow and continuous
* Logo animation: subtle and looped elegantly

### 7. Visual Consistency

Ensure the entire platform feels like one cohesive ultra-premium product.

* Color system

* Card styling

Every page can have a different cinematic background, but the overall identity must remain unified.

### 8. Accessibility and Usability

The design must remain usable, clear, and accessible.

Ensure:

* Text remains readable
* Contrast is strong enough
* Backgrounds do not interfere with content
* Animations do not distract users
* Important information remains visually dominant
* UI controls remain clear
* Motion feels premium, not decorative noise

Include support for reduced motion:

* Provide a simpler motion state for users who prefer reduced motion.
* Avoid essential information depending only on animation.

### 9. RTL and Arabic Compatibility

Check:

* Icon direction where needed
* Button motion direction
* Section reveal direction
* Navigation behavior
* Card spacing and layout
